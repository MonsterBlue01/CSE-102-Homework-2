Lab 3: ASCII-risks (Aterisks)

DESCRIPTION

In this lab, the users can put a normal positive integer in the program. Then they will get a 
pattern. That pattern will changed if users put different integer in.

INSTRUCTIONS

If user want to get a pyramid-like pattern with 5 rows, they just need to put 5. If they want to 
get it with another numbers of rows, just put that number in. But that number should be a
positive integer.
